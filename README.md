# 🌱 Hands Off, Villagers!

Villagers are cute.  
But when they start looting rare seeds like they're running a black-market flower ring — it's time to say **no**.  

Tired of farmers hoarding torchflower seeds and pitcher pods they can't even replant?  
*Hands Off, Villagers!* is a lightweight datapack that enforces healthy boundaries in your village.  
Because not everything green belongs in their grubby little hands.

---

## ✨ What it does

This datapack removes:
- `minecraft:torchflower_seeds`
- `minecraft:pitcher_pod`

from the `minecraft:villager_plantable_seeds` tag.  
This prevents villagers (especially farmers) from picking them up and stuffing them into their inventory forever.

---

## ✅ Features

- 🧠 **Vanilla-friendly** – no mods required  
- 🎮 **Client/server compatible** – drop into any world or server  
- 🎯 **Focused** – affects only two items  
- 🎨 **Custom icon included** – so even your datapack list looks great  
- 🆕 **Minecraft 1.21.2+ supported**

---

### 💡 Technical Notes

This datapack overrides:
`data/minecraft/tags/items/villager_plantable_seeds.json`
with a minimal seed list (excluding torchflower and pitcher pod).

If you're using other datapacks that edit the same tag, make sure to check compatibility or merge tag files accordingly.

# 🛠️ Installation and Technical Information

## Installation
- Make sure you have **Minecraft 1.21.2-1.21.8** with **NeoForge**, **Forge** or **Fabric** installed.
- **Fabric** requires **Fabric API** to be installed as well.
- Download the mod `.jar` file.
- Place it into your `mods` folder.
- Launch the game and enjoy your adventure!
